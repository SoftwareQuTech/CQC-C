Installation¶

Note

This library can be used to write applications which send CQC messages to a backend which can listen to these messages. This library does not come with a backend but you can use the simulator SimulaQron as a backend.

Install from source¶

To build from source (including tests), clone the repo, cd into the folder and do:

cd test
make all

To rum the tests, do:

./test_using_simulaqron.sh
